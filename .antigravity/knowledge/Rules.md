Правила створення бота:

1. Архітектура 100% як у попередніх ботах (core / actions / runner / user.settings / runner.config).
2. Python 3.11+, web3.py для EVM, python-telegram-bot (або aiogram) для Telegram.
3. Telegram-управління: команди /run, /stop, звіт після циклу, повідомлення про помилки.
4. Рандомізація (rng) + широкі delays між діями/гаманцями.
5. State.json — зберігає прогрес кожного гаманця.
6. wallets.txt — формат як раніше (name,0x... або просто 0x...).
7. Ніякого хардкоду ключів/токенів у коді.
8. Логи: кольоровий консоль + текстовий + JSON.
9. Anti-sybil: delays 30-180 сек між діями, 10-60 хв між гаманцями, cycles=1 за запуск.
10. Код чистий, з типами (typing), коментарями українською.

Кожен файл створюй повністю, з усіма імпортами. Не пиши "додай пізніше".
